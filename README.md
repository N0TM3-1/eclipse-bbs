# Eclipse BBS

A retro-style Bulletin Board System (BBS) designed specifically for Hack Clubbers, featuring classic ANSI aesthetics and community-focused features.

## 🌟 Overview

Eclipse BBS is a modern implementation of classic BBS systems, providing a nostalgic yet functional space for Hack Clubbers to connect, share, and engage with each other. Built on MysticBBS, it aims to recreate the golden age of bulletin board systems while serving as a unique hangout spot for the Hack Club community.

## 🔧 Technical Details

- **Platform:** MysticBBS (Version 1.12 A48)
- **Operating System:** Debian 12
- **Connection:** Telnet (Port 23)

## 🚀 Features

### Core Features (MVP)
- User verification system
- Customizable user account settings
- Mail system
- Discussion forums
- Door games integration

### Future Enhancements
- Message network creation and integration
- Additional community features

## 🔐 Security

The system implements various security measures including:
- User level permissions
- Password policies
- Ban/blacklist system
- Comprehensive audit logging

## 🎮 Entertainment

Door games are available with:
- Single and multiplayer options
- Scoreboard tracking
- Community competitions

## 🛠 Administration

SysOp tools include:
- System monitoring capabilities
- Moderation interface
- User management features

## 🎯 Target Audience

Eclipse BBS is specifically designed for Hack Clubbers, providing a unique space that combines retro computing aesthetics with modern community needs.

## ⚡ Getting Started

To connect to Eclipse BBS:
1. Use a Telnet client (for example [SyncTERM](https://syncterm.bbsdev.net/))
2. Connect to port 23 (server link coming soon)
3. Follow the new user registration process
4. Complete the verification questionnaire
